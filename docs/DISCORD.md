# 🎮 Discord Integration

Commonly's Discord integration creates a seamless bridge between Discord servers and Commonly pods, enabling bidirectional communication, automated summaries, and intelligent community management.

## 💾 **Database Storage Architecture**

### Message Storage Strategy
- **commonly-bot messages**: Stored in **PostgreSQL** (primary) for consistency and persistence
- **User synchronization**: Bot user automatically synced to PostgreSQL users table  
- **Message persistence**: Discord integration messages persist after browser refresh
- **Graceful fallback**: Falls back to MongoDB if PostgreSQL connection fails

### Technical Implementation
```javascript
// Bot message flow
1. Discord integration triggered
2. commonly-bot user synced to PostgreSQL (one-time)
3. Message stored in PostgreSQL with proper user join
4. Real-time broadcast via Socket.io
5. Message persists across page refreshes
```

## 🤖 **Bot Architecture**

### **Command Handling**
```javascript
// Discord interaction processing
router.post('/interactions', async (req, res) => {
  // Verify Discord signature for security
  if (!verifySignature(req)) {
    return res.status(401).json({ error: 'Invalid signature' });
  }
  
  const interaction = JSON.parse(req.body.toString('utf8'));
  
  // Handle ping (Discord endpoint verification)
  if (interaction.type === 1) {
    return res.json({ type: 1 });
  }
  
  // Handle slash commands
  if (interaction.type === 2 && interaction.data?.type === 1) {
    const { guild_id, data } = interaction;
    
    // Find integration for this guild
    const integration = await Integration.findOne({
      'config.serverId': guild_id,
      type: 'discord',
      isActive: true
    });
    
    if (!integration) {
      return res.json({
        type: 4,
        data: {
          content: '❌ Discord integration not found. Please install the bot first.',
          flags: 64 // EPHEMERAL
        }
      });
    }
    
    // Process command
    const discordService = new DiscordService(integration._id);
    const result = await discordService.handleInteraction(interaction);
    
    return res.json(result);
  }
});
```

### **Command Service Architecture**
```javascript
// services/discordCommandService.js
class DiscordCommandService {
  constructor(guildId) {
    this.guildId = guildId;
    this.integration = null;
  }
  
  async handleSummaryCommand() {
    // Get latest pod summary from Commonly database
    const latestSummary = await Summary.findOne({
      type: 'chats',
      podId: this.integration.podId
    }).sort({ createdAt: -1 });
    
    return {
      success: true,
      content: this.formatSummaryForDiscord(latestSummary)
    };
  }
  
  async handlePushCommand() {
    // Fetch Discord messages from last hour
    const messages = await this.fetchRecentMessages(50);
    
    // Create summary
    const summary = await this.createDiscordSummary(messages);
    
    // Post to Commonly pod via bot
    const botService = new CommonlyBotService();
    const result = await botService.postDiscordSummaryToPod(
      this.integration.podId,
      summary,
      this.integration._id
    );
    
    return result;
  }
}
```

## 📊 **Activity Aggregation**

### **Webhook Listener System**
```javascript
// Automatic Discord activity aggregation
class DiscordActivityAggregator {
  async aggregateHourlyActivity() {
    const activeIntegrations = await Integration.find({
      type: 'discord',
      'config.webhookListenerEnabled': true,
      isActive: true
    });
    
    for (const integration of activeIntegrations) {
      try {
        // Fetch Discord messages from last hour
        const messages = await this.fetchChannelMessages(integration);
        
        if (messages.length > 0) {
          // Create AI summary
          const summary = await this.createActivitySummary(messages);
          
          // Post to Commonly pod
          await this.postSummaryToPod(integration.podId, summary);
        }
      } catch (error) {
        console.error(`Failed to aggregate activity for ${integration._id}:`, error);
      }
    }
  }
}
```

// Process Discord messages for AI analysis
const processDiscordMessages = (messages) => {
  return messages
    .filter(msg => !msg.author.bot)  // Exclude bot messages
    .filter(msg => msg.content.length > 5)  // Meaningful content only
    .map(msg => ({
      username: msg.author.username,
      content: msg.content,
      timestamp: new Date(msg.timestamp),
      reactions: msg.reactions?.length || 0
    }))
    .sort((a, b) => a.timestamp - b.timestamp);
};

class CommonlyBotService {
  async postDiscordSummaryToPod(podId, summary, integrationId) {
    // Get or create bot user
    const botUser = await this.getOrCreateBotUser();
    
    // Format message for Commonly
    const message = this.formatDiscordSummary(summary);
    
    // Post to pod via Socket.io
    const io = getSocketInstance();
    io.to(`pod_${podId}`).emit('message', {
      user: botUser,
      content: message,
      timestamp: new Date(),
      type: 'discord_summary'
    });
    
    // Save to database
    await this.saveMessageToDatabase(podId, botUser._id, message);
    
    return { success: true };
  }
}

### **Signature Verification**
```javascript
// Verify Discord request authenticity
function verifySignature(req) {
  const signature = req.headers['x-signature-ed25519'];
  const timestamp = req.headers['x-signature-timestamp'];
  const body = req.body.toString('utf8');
  
  if (!signature || !timestamp || !DISCORD_PUBLIC_KEY) {
    return false;
  }
  
  const signatureBytes = Buffer.from(signature, 'hex');
  const publicKeyBytes = Buffer.from(DISCORD_PUBLIC_KEY, 'hex');
  const message = Buffer.from(timestamp + body, 'utf8');
  
  return nacl.sign.detached.verify(message, signatureBytes, publicKeyBytes);
}
```
